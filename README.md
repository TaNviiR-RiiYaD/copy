- Mask URL support

  - Ngrok (With or without hotspot)


### Installation

- Just, Clone this repository -
```
$ git clone git://github.com/TaNviiR-RiiYaD/copy
```

- Change to cloned directory and run `zphisher.sh` -
```
$ cd zphisher
$ bash zphisher.sh
```
